# Visas and Immigration

 Taiwan offers Visa-Exempt Entry to many number of countries. In most cases, you should be able to visit Taiwan without any Visas or if a visa is requried, Visitor visa would be sufficient.

## Check if you need visa to visit Taiwan

Check out [Visa-Exempt Entry](https://www.boca.gov.tw/cp-149-4486-7785a-2.html) page from the Bureau of Consular Affairs website to see if you can visit Taiwan without any visa.

If your case is uncommon therefore difficult to check with the link above, You may also use [TravelDoc](https://www.traveldoc.aero/) or [IATA TravelCentre](https://www.iatatravelcentre.com/passport-visa-health-travel-document-requirements.htm) to check what are required for you to visit Taiwan.

## Types of Visa (or other entry permit application)

If you are not eligible for visa-exempt entry, You will need to apply for a visa (or other entry permit application).

### Travel Authorization Certificate

If you met following conditions, you can apply for "Travel Authorization Certificate" online instead of getting a visa.

- You are a citizen of India, Vietnam, Indonesia, Myanmar, Cambodia or Lao. 
- You have already been to Taiwan, United States, Canada, the United Kingdom, any of EU and Schengen countries, New Zealand, Australia, Japan or Korea in recent years and have valid documents that can confirm that.

Visit [this link](https://niaspeedy.immigration.gov.tw/nia_southeast/) to apply for Travel Authorization Certificate. You will find the detailed requirements during the application process.

### Visitor Visa for Tourism Purpose

Most people not eligible for visa-exempt status and also Travel Authorization Certificate would be sufficient with this option. [Check out the requirements here](https://www.boca.gov.tw/cp-158-250-a5a40-2.html) then start your visa application [with this link](https://visawebapp.boca.gov.tw/BOCA_EVISA/).

### Visitor Visa for Attending Conference / Exhibition

If you are not eligible for "Visitor Visa for Tourism Purpose". For example, if you are a citizen from countries listed on [this page on BOCA website](https://www.boca.gov.tw/cp-159-249-4162c-2.html). You will need to apply with this option.

[Check out this link](https://www.boca.gov.tw/cp-158-259-e6930-2.html) to learn more about this option.

**Note: This option requires an invitation letter**

### Entry permit for people from China, Hong Kong, or Macau

If you are from China, Hong Kong, or Macau, [Visit this link](https://coa.immigration.gov.tw/coa-frontend/overseas-foreign-china) to apply for entry permit.

## Requesting Visa invitation letter

If you met **ALL** followings, We can help with providing visa invitation letter for you.

- You are not eligible to enter Taiwan using **visa-exempt status**, **a tourist visa ([Visitor Visa for Tourism Purpose](https://www.boca.gov.tw/cp-158-250-a5a40-2.html))** or [Travel Authorization Certificate](https://niaspeedy.immigration.gov.tw/nia_southeast/).
- You need an invitation letter to apply for a [**Visitor Visa for Attending Conference / Exhibition**](https://www.boca.gov.tw/cp-158-259-e6930-2.html) at a Taiwan diplomatic office.

[Visit the link to see guidance on how you can request one then apply for visa.](https://hackmd.io/@coscup/ry2VoMnqWl)

## Online Arrival Card

Before arriving in Taiwan, You're also required to submit Online Arrival Card 3 days within the arrival day. [Visit this link to submit arrival card](https://twac.immigration.gov.tw/)

**Note: This is requried for all visitors including travelers with visa-exempt status.**
